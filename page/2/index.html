第三章：政府投融资与债务

发表于 2022-10-16 分类于 读书笔记

上一章提到政府的债务，本章对其详细叙述。再穷的国家也有大片的土地，土地本身并不值钱，值钱的是土地之上的经济活动。经济发展的奥秘之一，正是把有形资产转本成为抽象资本，从而聚合跨越空间和时间的资源。

我国政府不但拥有城市土地，也掌控着金融系统，自然会以各种方式参与实业投资，不可能置身事外。法律规定，地方政府不能从银行贷款，所以政府想要借钱投资，需要成立专门的公司。这类公司大都是国有独资企业，一般统称为“地方政府融资平台”，突出了其融资和负债功能。

第二章：财税与政府行为

发表于 2022-10-16 分类于 读书笔记

这一章通过两部国产电视剧《大明王朝1566》和《走向共和》引出，正好这两部电视剧我都有一点点看完，便也理解其实所有的政见冲突和人事谋略，都隐藏在账目中。就像书中说道，要真正理解政府行为，必然要了解财税。

从上一章了解到政府的事权划分，而事权必然要求相应的财力支持，财政是国家治理的基础和重要支柱。中国在20世纪80年代的经济特点就是“承包”，而我国的基本国策决定了不能对所有权做出根本性变革，只能对使用权和经营权实行承包制，以提高工作积极性。

第一章：地方政府的权力与事务

发表于 2022-10-16 分类于 读书笔记

读完第一章，让我对中国地方政府有了更加深刻的认识。正如作者所说，在中国脱离政府谈经济，是完全不可能的。而要了解政府的治理和运作，则需要了解权力和资源在政府体系中的分布规则。

中国的五级政府管理体系为中央—省—市—县区—乡镇，又因为现实情况远比该简化的体系复杂，所以这五级体系也在不断改革中。中国是如此之大，为了维持大一统的国家，则要求维护中央权威和统一领导；为了政治体系的日常运作，又需要以地方政府为主。因此，地方政府自主权力很大。该章节中讲述了对事权划分的三种理论：公共服务的规模经济与边界、信息复杂性、激励相容。
