Faithful Reasoning Using Large Language Models

发表于 2022-10-17 分类于 论文精读

Methodology

comprises two fine-tuned LMs, one for selection and one for inference. The interleaved operation of these two components has a causal structure.

Selection-Inference:

Selection model chooses a set of statements from the context. (selection)

Inference model predicts an entailment by computing a statement that follows from the selection. (inference)

Halter:

looks at the output of a Selection-Inference step and determines if there is sufficient information to answer the question.

it cannot rely on knowledge embedded in its weights, but must depend on the reasoning trace.

Two-Stage Halter:

  1. predict whether the question can be answered given the current inference and the question.
  2. computes an answer given the final inference and minimal additional information.

Value Function($LM_{value}$):

perform a step-level beam search on the reasoning traces to find the best candidate for answering the question.

Datasets

Proof Writer and a question-answering version of EntailmentBank

第六章：债务与风险

发表于 2022-10-16 分类于 读书笔记

在大规模城市化过程中，地方政府背上了沉重的债务，地价和房价飞涨，也让居民背上了沉重的债务。人们在乐观时往往会低估负债的风险，过多借债。

经济的正常运行离不开债务，但负债率搞得经济中，资产价格的下跌往往迅猛；资产价格下跌会引起信贷收缩，导致资金链断裂。一个部门的负债就对应着另一个部门的资产，债务累积或“加杠杆”的过程，就是人与人之间商业往来增加的过程，会推动经济繁荣；而债务紧缩或“去杠杆”也就是商业活动减少的过程，会带来经济衰退。债务带来的经济衰退还会加剧不平等，即债务常常把风险集中到承受能力最弱的穷人身上。

第三章：政府投融资与债务

第一章：地方政府的权力与事务

发表于 2022-10-16 分类于 读书笔记

读完第一章，让我对中国地方政府有了更加深刻的认识。正如作者所说，在中国脱离政府谈经济，是完全不可能的。而要了解政府的治理和运作，则需要了解权力和资源在政府体系中的分布规则。

中国的五级政府管理体系为中央—省—市—县区—乡镇，又因为现实情况远比该简化的体系复杂，所以这五级体系也在不断改革中。中国是如此之大，为了维持大一统的国家，则要求维护中央权威和统一领导；为了政治体系的日常运作，又需要以地方政府为主。因此，地方政府自主权力很大。该章节中讲述了对事权划分的三种理论：公共服务的规模经济与边界、信息复杂性、激励相容。
