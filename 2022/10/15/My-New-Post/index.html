Error Propagation：一步错，步步错

Decoder

​ Self-attention变为Masked Self-attention

​ 预测出则decoder停止

Non-Autoregressive（NAT）：decoder一次性结束

Cross attention![](D:\MasterNote\image\Cross attention.JPG)

Transformer中的Decoder不一定要用Encoder的最后一层作为输入，只是原始paper是这样的。

Teacher Forcing：训练的时候，给decoder输入正确的答案，即Ground Truth。这样预测的时候会出现Mismatch，即成为exposure bias。

Scheduled Sampling：在训练的时候不要给decoder所有正确的输入，适当给一些错误的信息，解决exposure bias的问题。

BLEU score不能微分，所以不能作为训练的目标，一般作为评价指标。或者使用Reinforcement Learning来解决这个问题。

Pre-train Model

Represent each token by a embedding vector

Word2Vec， Glove

FastText：每次输入英文的一个字符到模型中，最后得到这个单词的embedding

Contextualized Word Embedding

Tree-based Model效果不是很好，在处理文法结构非常清楚的时候效果很好（比如数学公式）

一般都是用LSTM，Self-attention

Smaller Model

Network Pruning， Knowledge Distillation， Parameter Quantization， Architecture Design

Network Architecture

Transformer-XL：使模型能够输入的长度更长

Reformer， Longformer：减少Self-attention的计算复杂度

How to fine-tune

输入为多个句子时，在两个句子中加入[SEP]，输入到模型中。

输出一个class：将整个句子有关的Embedding放到一个模型中

每个token输出一个class：将每个token对应的Embedding输入到模型中

从输入中做copy(Extraction-based QA)：存在两个向量，分别跟document中每个token做dot product，最后通过softmax的结果，分别确定要复制的起点和终点

输出一个句子：Seq2Seq Model，将encoder中对应的Embedding输入到decoder中，再将decoder每次输出的结果重新输入到encoder中，继续将encoder的输出结果输入到decoder中，最后输出结束。

可以将pre-train model固定住，只训练自己的模型；

也可以同时将pre-train model和自己的模型作为一整个模型进行训练。

Adapter：整个模型进行训练时，只将pre-train model中的部分参数进行训练，只会修改Adapter Layer的参数

Weighted Features: 将每层的feature进行 weighted sum，其中的权重w可以学出来。

Predict Next Token

Language Model：LSTM等

Self-attention：需要设置限制，在每个位置只能看到前面的token

ELMo：双向LSTM，考虑了上文和下文，但是不是同时。

BERT：将一个句子中的token，进行MASK或者随机换成其他的token，可以同时看到周围的信息。与CBOW的训练方法一模一样，只是BERT可以看到的长度无限、模型更复杂。

Masking Input

**Whole Word Masking(WWM)**：一个盖住一整个word

Phase-level & Entity-level(ERNIE)

SpanBert：一次盖住一排token。

Span Boundary Objective(SBO)：将盖住部分的左边和右边的信息输入，并给出一个数值，即需要预测的token序号(第几个token)。比较适合coreference

XLNet：Transformer-XL

根据sentence中一部分来predict出mask的地方，不让模型看mask，只让他看position的信息。

autoregressive model：生成一个句子的时候是由左到右生成的，decoder每次都要输入上一次时间点的token才能输出下一个时间点的token。(BERT不太擅长)

MASS/BART：预训练一个pre-train model，在encoder对输入进行一定程度的破坏，让decoder重新输出原始输入

MASS：把一些部分random mask/delete/permutation/rotation/text infilling，只要decoder能还原出mask的部分

text infilling(效果最好)：可能mask多个token，或者没有token但依然放一个mask

UniLM：一个model可以是encoder，也可以是decoder，同时也可以是encoder+decoder

Electra：将sentence中的其中一些token随机替换成其他的token，让模型分辨出哪些token被置换（分类任务），每一个token都会有对应的输出。

用一个比较小的BERT（train的不好）将一些token置换为其他的token。

Sentence Level

使用一个vector代表整个sentence

Skip Thought：将一个句子输入到encoder中，使用decoder预测出下一个句子

Quick Thought：分别将两个句子输入到encoder中得到两个vector，如果这两个句子相近，则让这两个vector越相近越好。

BERT中，[CLS]代表整个句子的vector。使用NSP任务，将[CLS]的vector输入到二分类器中，判断后面的两个句子是不是相邻的。（可能任务比较简单，所以效果不好）

ALBERT中，使用SOP任务，将两个句子相接，模型输出yes；将两个句子位置互换，模型输出no

BERT+外部知识=ERNIE

Non-autoregressive Sequence

出现的问题是，decoder的每一层的输出结果与之前网络层的输出结果是相互独立的，输出结果会有多个可能。

Vanilla NAT：在encoder中每一个token都会输出一个数字，即在的decoder中对应需要输出几个token，这样在decoder中需要copy encoder的输入作为decoder的输入。

Knowledge distillation：首先把input输入到大model（teacher）中得到一个概率分布，然后再把input输入到小model（student）中，让student的概率分布与teacher的尽可能接近

这里teacher是auto-regressive model，student是non-autoregressive model

Noisy Parallel Decoding（NPD）：通过NATsample出来各种不同的数字，得到多个不同的结果，再将这些结果输入到Autoregressive Decoder Teacher来打分数，选一个最好的句子即可。（这里autoregressive只需要一个step就行）

Domain Adaptation

Domain shift：训练和测试数据存在不同的分布

（可以看成是Transfer learning的一种）

​

​ 如果target domain中有很少但是已经标注好的数据，可以用它们来微调用source domain训练好的模型。（不能跑太多的iteration，防止overfitting）

​ 如果target domain中有很多并且没有标注好的数据，找一个Feature Extractor（network），让它从Source和target中分别抽出相同分布的两个feature。
